# 安装

```shell
cnpm install -g create-react-app
```

# 创建项目

```shell
create-react-app xxxx
```

# 命令

## 启动

```shell
yarn start
```

## 打包

```shell
yarn build
```

## 测试

```shell
yarn test
```

## 暴露全部webpack配置文件到项目中(不可逆)

```shell
yarn eject
```

# 目录解构

- **public**   模板静态文件夹
  - **favicon.ico**                浏览器tab图标
  - **index.html**                 模板
  - **logo.png**                    react logo
  - **manifest.json**            安卓/ios的壳配置文件
  - **robots.txt**                  爬虫协议
