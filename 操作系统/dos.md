# 常用命令

## 目录操作

### 创建文件夹

```
md 文件夹名称
```

### 切换文件夹

```bash
C:
```

### 查看所有目录

```bash
dir
```

### 切换目录

```bash
cd 目录名
# 跨盘符切换
cd /d D:目录名
```

### 删除目录

```
rd 目录名
```



## 文件操作

### 创建文件

```bash
cd>文件名
```

### 删除文件

```bash
del 文件名
```

## 终端

### 清空屏幕

```bash
cls
```

### 退出终端

```bash
exit
```

## 网络

### 查看ip

```bash
ipconfig
```

### 测试ip

```bash
ping ip地址
```

- route -p  add 0.0.0.0 mask 255.255.0.0 192.168.3.1

## 其他

### 修复虚拟机

部分情况, 可能导致虚拟器无法启动的情况, 后台显示一直被占用, 使用如下命令

```
netsh winsock reset
```
