# 常用命令

## 目录操作

### 创建文件夹

```
md 文件夹名称
```

### 切换文件夹

```bash
C:
```

### 查看所有目录

```bash
dir
```

### 切换目录

```bash
cd 目录名
# 跨盘符切换
cd /d D:目录名
```

### 删除目录

```
rd 目录名
```



## 文件操作

### 创建文件

```bash
cd>文件名
```

### 删除文件

```bash
del 文件名
```

## 终端

### 清空屏幕

```bash
cls
```

### 退出终端

```bash
exit
```

## 网络

### 查看ip

```bash
ipconfig
```

### 测试ip

```bash
ping ip地址
```

## 打开软件

### 计算器

```bash
calc
```

#### 画图工具

```
mspaint
```

### 记事本

```
notepad
```
