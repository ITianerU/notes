## 运算符

### 位运算

```
  A = 0011 1100
  B = 0000 1101
A&B = 0000 1100  都为1时为1
A|B = 0011 1101  有1为1
A^B = 0011 0001  只有一个1时为1
 ~B = 1111 0010  1变0， 0变1
 
 << 左移 0001 变为 0010 相当于*2; so  2<<3 2左移三位 0010 -> 10000 为16
 >> 右移 相当于 /2   
```
